# Starhills26JK

친구들이 집 방문 일정을 선택하고 방문 정보를 남길 수 있는 정적 웹 서비스입니다.

## Features

- 방문 가능 날짜와 시간 선택
- 숨겨진 관리자 모드로 운영 규칙 관리
- 공휴일, 예약 불가, 예약 완료 날짜 설정
- 안내 문구와 교통수단별 주의사항 편집
- 방문 확정 후 Google Calendar 저장 링크 제공

## Run Locally

```bash
python3 -m http.server 5174
```

Then open:

```text
http://localhost:5174
```

## Deployment

This project is a static site and can be deployed directly to Vercel.
